The basics of graphics in HTML

Non-text in web pages
  • Colors - Text, background
    This example is what sets the background color and text color of this page: 
    <BODY TEXT="#440000" BGCOLOR="EBC79E">

    Here is some Purple Text: 
    Here is some<FONT COLOR="#CD09C2">Purple Text</FONT>

    For a table of Hex codes for colors, see the Hexadecimal Color Chart or the first section of Background Colors, or Color mapping.

  • Images - In place, background
    Here is the code that creates this line: 
    <img border=0 align="bottom" src="Palette.gif" alt="Palette icon">Here is the code...
  (Note that the "alt" tag is used to describe what the picture is in case it doesn't load.)

    This is an example of including a graphic as a background image: 
    <BODY TEXT="#440000" BACKGROUND="../tan_paper.gif">

    Different kinds of images (see Gordon's Overview of Web Graphics for more details)
    • GIF - Good compression, better for simple art and line drawings
    • JPEG - more compression options, better for photographic items
    • animated GIF - cartoon-like animations of GIF images (see bat at bottom of page)
  • Rules and Icons
    The basic Horizontal Rule is <hr> (as shown as basic dividers in this page), but it can be specified with alignment and size as well: 
    <hr align=center size=7 width=200>
    Sometimes a long slender graphic works well as a horizontal rule
  • Sounds and Video
    There are a number of ways of including sounds and video in web pages, but Quicktime is fairly easy and usually is available from most browsers. Here is an example of including a Quicktime sound and video clip. In these examples a trick is included to force the sound or movie to be played by a QuickTime plugin. A dummy movie file must exist and be in the directory for this to work. 
     <embed src="dummy.mov"
width=200 height=60 showcontrols=true
autostart=false qtsrc="filename.mov">

     <embed src="dummy.mov"
width=200 height=300 showcontrols=true
autostart=false qtsrc="filename.mov">
Placement
  • Simple alignment
    This is simply placed in the code along with text. Here is the code that created it: 
    <img border=0 align="bottom" src="Palette.gif">This is simply...

    Align="top" (bottom is default)

    Align="center"

    Align="left" (the default)  
     

    Align="right"    look way over there - - - - - - - - - - - - - - - ->
    Note that some subsequent lines get included in the alignment. if you do not want them aligned with the image, you will need to include some blank lines in the code.

  • Tables
    Tables are nice for aligning images and text. This example has borders, but they may be set to 0 if you do not want the look of a grid. 
    <table width=600 border=3>
<tr>
<td>Tables are nice for aligning images and text.
This example has borders, but they may be set to 0 
if you do not want the look of a grid.</td>
<td valign="top"><img border=0 src="Palette.gif"></td>
</tr>
<tr>
<td>(the stuff that goes in the lower left
part of this table :-)</td>
<td valign="bottom">This is the code for generating this table.
You can really make things interesting by including
tables within tables.</td>
</tr>
</table>
    		 This is the code for generating this table. You can really make things interesting by including tables within tables.
  • Frames
    Frames are used in may web pages to give separate functionality to different areas of the window. This example does not use frames. Frames are complicated and are not covered in this class.
