HTML Basics (WWW part 2)

Basic structure of the language

Commands are of the form: < command > affected text </command >. NOTE: capitalization is optional.

Here is the most simple HTML file:

<HTML>
<HEAD>
<TITLE>First Try</TITLE>
</HEAD>
<BODY>
Hello World!
</BODY>
</HTML>

The point of the HTML commands is to specify the formatting of the text and to provide for links to other Web things and to embed other kinds of things in the document.

Some basic formatting codes:

(The first three do not need closing codes, the rest do)

<p> - paragraph break (If you don't put one in, paragraph breaks will be ignored, running everything together).
<br> - line break - just like a paragraph break, except without the extra white space between paragraphs
<hr> - horizontal rule - draw a thin line, start a new line. (See below.)

<Center>Centered Text</Center>

<B>BOLD</B> <I>italic</I> <U>underline (not always implemented)</U>

<TT>Typewriter (mono-spaced)</TT>

<PRE>

      . Preformatted (mono-spaced)  .
      .               Text          .

</PRE>

Lists:

<BLOCKQUOTE>

apples

oranges

bananas

</BLOCKQUOTE>

<UL>

  • apples (actually: <LI>apples)
  • oranges (actually: <LI>oranges)
  • bananas (actually: <LI>bananas)

</UL>

<OL>

  1. apples (actually: <LI>apples)
  2. oranges (actually: <LI>oranges)
  3. bananas (actually: <LI>bananas)

</OL>

Also - definition lists (See bare-bones document for details).

Images (an example):

<IMG border="1" width="98" height="140" ALIGN=bottom SRC="Me.gif" ALT = "My Picture Here">

Anchors:

<A HREF="URL "> anchored text </A> (Link to a document)

<A HREF="URL "> <IMG SRC="anchoredImage.jpg"> </A>

<A NAME="Top"></A> (Link to a place within this document - the beginning) referred to by:

<A HREF="#Top"> Go To the Top</A>
